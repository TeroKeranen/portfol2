Tero Keränen

Web ohjelmoinnin opiskelija

Aloitin Helsingin Taitotalossa 2021 marraskuussa ohjelmistokehittäjä, web kehitys puolella.

Hei!

Kiinnostukseni ohjelmoinnista lähti aikoinaan kun aloitin itsenäisesti harjoittelemaan Pythonin parissa, ennen opintoja myös harjoittelin HTML ja CSS parissa, jonka johdosta sainkin idean hakea Taitotalon opintoihin mukaan.
Tällä hetkellä vahvinta osaamistani ovat HTML, CSS, Nodejs ja Javascript. Kiinnostunein olen Javascriptistä, jota harjoittelenkin itsenäisesti todella paljon. Tututksi ovat myös tulleet koulutuksen myötä Bootstrap, React sekä hieman Angular.
Tietokannoista tutuiksi on tullut MySQL johon olen koulun kautta harjoitellut sekä Mondodb joka on tullu tutuksi Nodejs projektien kautta.
